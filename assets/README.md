# Assets & Attributions

This directory contains image assets used for documentation and reference purposes only.

These assets are NOT used directly by the application at runtime.
All icons referenced by the Discord Rich Presence must be uploaded manually
to the Discord Developer Portal.

---

## Asset Usage Policy

- Assets in this folder are included for documentation, preview, and attribution
- The application does not load or distribute these files programmatically
- Runtime usage relies exclusively on Discord Developer Portal asset IDs
