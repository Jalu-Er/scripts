# Discord RPC Integration

This directory contains implementations for Discord Rich Presence (RPC) integration for different operating systems.

You can find specific versions and their respective tutorials/documentation in the following subdirectories:

*   **discord_RPC_linux**: For Linux-based systems. Refer to `discord_RPC_linux/README.md` for setup instructions.
*   **discord_RPC_windows**: For Windows-based systems. Refer to `discord_RPC_windows/README.md` for setup instructions.
